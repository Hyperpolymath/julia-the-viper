================================================================================
                        SECURITY GUARANTEES
================================================================================

Regardless of which license you choose, Julia the Viper provides the following
architectural security guarantees:

  1. Code injection is grammatically impossible
  2. Data expressions are provably total (guaranteed to halt)
  3. Pure functions cannot have side effects in Data context
  4. Number operations are overflow-protected

These guarantees are enforced by the language grammar itself, not by runtime
checks. See SECURITY.md for details.
